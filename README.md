# MERN CRUD App

A full-stack MERN CRUD application built using:

- React.js
- Node.js
- Express.js
- MongoDB

## Features

- Create data
- Read data
- Update data
- Delete data

## Tech Stack

### Frontend
- React.js
- Axios

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose

## Installation

### Clone Repository

```bash
git clone https://github.com/YaramasuBhavya/mern-crud-app.git
```

### Backend Setup

```bash
cd backend
npm install
npm start
```

### Frontend Setup

```bash
cd frontend
npm install
npm start
```
